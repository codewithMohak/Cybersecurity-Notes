## **Steps to configure the Foxy Proxy in the specific browser that is Firefox**

### Step 1: *Download and Install the Foxy Proxy Basic Extension*

### Step2 : Access Foxy Proxy Options : *Once Install, a button will appear in the top right of your browser*

### Step3 : Create Burp Proxy Configuration : *Create a new Foxy Proxy by clicking add button*

### Step4 : Add Proxy Details: *Fill the following values*
- **Title** : Burp
- Proxy IP : 127.0.0.1
- Port : 8080
### Step5 : Save the configuration

### Step6 : Activate Proxy Configuration : *Click on the Foxy Proxy Icon on the top right and select the `Burp` or whatever title you gave and this will browser the traffic through  `127.0.0.1:8080` * .

### Step7 : Enable Proxy Configuration : *Make sure your interception is turned on in the Proxy tab*.

### Step8 : Test the Proxy : *Open Firefox and try to accessing the website and your browser will hand and your proxy tab will populate with the HTTP requests*.
